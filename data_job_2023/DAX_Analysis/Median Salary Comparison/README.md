# 🌎 Median Salary Comparison Analysis – US vs Non-US vs Global

## 📊 Project Overview
This analysis compares **median salaries across three categories**:
- **United States**
- **Non-United States**
- **Overall (Global Median)**

The dataset was **cleaned and transformed using Power Query** and analyzed using **Power Pivot** for accurate aggregation.  
The results are visualized using a **Stacked Column Chart**, allowing users to easily compare salary distributions while dynamically filtering data using a **country slicer**.

This project demonstrates practical **Excel-based data analysis and visualization skills** relevant to a Data Analyst role.

---

## 🎯 Business Objective
The objectives of this analysis are to:
- Compare **median salary levels** between US and non-US job markets
- Understand **global salary distribution**
- Enable **interactive country-based filtering**
- Support **data-driven compensation and workforce insights**

---

## 🛠 Tools & Technologies
- **Microsoft Excel**
  - Power Query – Data cleaning and transformation
  - Power Pivot – Data modeling and calculations
  - Pivot Tables & Pivot Charts
  - Slicers – Country-level filtering
- Data Analysis & Reporting

---

## 📊 Data Preparation & Modeling
1. Imported and cleaned job market data using **Power Query**
2. Standardized country and salary fields
3. Built a **Power Pivot data model**
4. Created measures to calculate:
   - US Median Salary
   - Non-US Median Salary
   - Overall Median Salary
5. Enabled interactive filtering using a **Country slicer**

---

## 📈 Visualization Details
- **Chart Type:** Stacked Column Chart
- **X-Axis:** Salary Category (US, Non-US, Global)
- **Y-Axis:** Median Salary
- **Slicer:** Country

The visualization provides a clear comparison of **salary differences across regions**.

---

## 🎛 Interactivity
- Users can filter data using the **Country slicer**
- Chart updates dynamically based on selected countries
- Enables flexible, **self-service exploration** of salary trends
